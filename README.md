# Todos Tailwind

A tailwind + react-query + typescript experiment [(see it live)](https://gallant-pike-f5fb05.netlify.app/)

## Getting started

Checkout the project, then:

```
yarn install
yarn start
```

To build the app:

```
  yarn build
```

To serve the built locally:

```
  yarn serve
```

_Enjoy!_
